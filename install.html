(กำลังพัฒนา)

เปิดแอปพลิเคชันทันที

วิธีติดตั้งแอปไว้บนหน้าจอหลัก (Home Screen)

เพื่อความสะดวกในการเรียกใช้งานครั้งต่อไปโดยไม่ต้องเข้าผ่านเบราว์เซอร์

สำหรับ iOS (iPhone/iPad)

  • เปิดลิงก์ หรือสแกน QR Code ด้วยแอป Safari เท่านั้น
  • แตะที่ไอคอน แชร์ (Share) บริเวณแถบเมนูด้านล่างของหน้าจอ
  • เลื่อนหาแล้วแตะที่คำว่า เพิ่มไปยังหน้าจอโฮม (Add to Home Screen)
  • กดปุ่ม "เพิ่ม" (Add) ที่มุมขวาบน

สำหรับ Android

  • เปิดลิงก์ หรือสแกน QR Code ด้วยแอป Google Chrome
  • แตะที่ไอคอน เมนู (จุด 3 จุด) มุมขวาบนของหน้าจอ
  • เลือกคำว่า ติดตั้งแอป (Install app) หรือ เพิ่มลงในหน้าจอหลัก
  • กดยืนยัน "ติดตั้ง" (Install) แล้วรอสักครู่

เมื่อติดตั้งเสร็จแล้ว ไอคอน "น้อง AIDE SWD" จะไปปรากฏที่หน้าจอมือถือของคุณ เหมือนแอปพลิเคชันทั่วไปเลยครับ!
